1.一棵完全二叉树第六层有9个叶结点（根为第一层），则结点个数最多有（D）

```
A 112
B 111
C 107
D 109
```

:foolman

> 根据二叉树的性质，二叉树的第i层上的节点数最多2^i （i>=0,即第一层i=0），第六层的节点数最多为2^5 =32,第七层节点数=2*（32-9）,总数即数每层节点数相加，2^0 +2^1 +2^2+ ..+2^5+2*(32-9)=109
